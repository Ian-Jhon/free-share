# free-share

通过一条链接与他人分享本地的任何文件，基于 [js-psychokinesis](https://github.com/psychokinesis-dev/js-psychokinesis) 构建。

## 构建
```bash
$ apt-get install nodejs npm
$ npm install
$ npm install -g bower
$ bower install
```

## 启动
```bash
$ npm start
```

## 打包
```bash
$ npm run-script pack
```

完成后各平台对应的可执行程序位于上层目录的 free-share-dist 下。
